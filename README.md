# TaskBloom

TaskBloom is an interactive to-do list web application developed using HTML, CSS, and JavaScript.
It allows users to efficiently manage daily tasks through a clean interface and responsive design.

---

## Features

* Add new tasks
* Add tasks using the Enter key
* Inline validation for empty input
* Mark tasks as completed
* Automatically move completed tasks to the bottom
* Delete tasks with animation
* Task summary showing total, pending, and completed tasks
* Smooth animations and transitions
* Sound effect when marking tasks as complete
* Data persistence using localStorage
* Responsive design for different screen sizes

---

## Technologies Used

* HTML5 – Structure of the application
* CSS3 – Styling, animations, and responsiveness
* JavaScript (Vanilla JS) – Application logic and interactivity
* Vercel – Deployment platform
